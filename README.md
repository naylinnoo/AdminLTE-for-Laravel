**AdminLTE for Laravel** -
This is Laravel Ready AdminLTE 2

## Documentation & Installation Guide


**Installation** 
- Go into To Copy file
- Copy the folders from "To public" to Laravel Public folder
- Copy the folders from "To view" to Laravel view folder


#### Requirements:

- This is currently working with Laravel version 5.8
- Not tested for future Laravel Versions (Eg Laravel 6)
